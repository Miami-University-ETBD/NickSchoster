About the Game

Bug Bouncers is a fast-paced action game where players control an insect inside a bubble.

The goal is to knock enemy insects into the grass to pop their bubbles while avoiding the same fate!

Game Mechanics

Feature Description
Movement Floaty, bouncy physics-based mobility allowing for high-speed maneuvering.
Combat Bounce into enemy insects to knock them into the grass.
Death If you touch the grass first, your bubble pops and the game restarts.
Win Condition Eliminate all enemy insects by bouncing them into the grass.
