The Development Process

Bug Bouncers was created as part of the 2025 Global Game Jam. Given a time limit of three days, I developed this game from start to finish.

Using Unity2D 2023, I built almost eveything myself. There were many fetures that I did not get to add due to the time limit, but they may make an apperience in the future.

Art & Code

Every visual element in Bug Bouncers was created by me. This includes the background artwork as well as the players and enemies. I designed them in a pixel art style.

On the coding side, I wrote all the scripts myself, which was hard considering the time limit.

Game Jam Experience

Participating in the game jam was an intense but rewarding experience. I learned a lot about rapid game development and balancing mechanics under the preasure of a tight deadline.
